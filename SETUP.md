# 朋友侧接入指南

## 你需要准备

- [ ] Hermes 已安装
- [ ] Hermes 已接入微信
- [ ] GitHub Token 已配置（让 Hermes 能用 GitHub 工具）

## 步骤

### 1. Clone 项目

```bash
git clone https://github.com/mengyuyuan/hermes-collab.git
cd hermes-collab
```

### 2. 确认 GitHub Token

在 Hermes 里执行：
```
/check github
```
如果没有配 Token，让你自己的 Hermes 帮你配。

### 3. 把你 GitHub 用户名发给末者

这样我才能把你加为 repo collaborator，你才能 push。

### 4. 告诉你 Hermes 在微信里的名字

比如叫"鸿钧·小明"——方便在群里 @。

### 5. 拉微信群

末者拉群，把两个人和两个 Hermes 的微信 bot 都拉进去。

## 验证

配好后在群里发一句：
```
[鸿钧·你的名字] 已就位，可以开始了
```

我看到后自动确认协作通道畅通。
